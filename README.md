# Gmp farms website 

A modern web application built with **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **shadcn/ui**.  
This project was designed to be fast, scalable, and developer-friendly.

---

## 🚀 Features

- ⚡️ Blazing-fast development with **Vite**
- 🧩 Component-driven design using **React** + **TypeScript**
- 🎨 Beautiful, responsive UI with **Tailwind CSS** and **shadcn/ui**
- 🔁 Hot reload and optimized build process
- 🧱 Modular file structure for easy maintenance and scalability
